# Googleスプレッドシート連携の準備

このファイルは、透析管理アプリの記録をGoogleスプレッドシートへ保存するための手順です。

## 目的

- 透析記録と飲水記録をGoogleスプレッドシートへ保存する
- 将来作る食事管理アプリから、飲水量などを読み取れるようにする
- 食事管理アプリからは読み取りだけにする

## 作るシート

GASが自動で次の2つのシートを作ります。

- 透析記録
- 飲水記録

## 手順

1. Googleスプレッドシートを新しく作ります。
2. メニューの「拡張機能」から「Apps Script」を開きます。
3. `gas-dialysis-api.gs` の中身をApps Scriptへ貼り付けます。
4. Apps Scriptの「プロジェクトの設定」で、スクリプトプロパティを追加します。
5. `WRITE_TOKEN` に自分だけが分かる文字を入れます。
6. 必要なら `READ_TOKEN` に読み取り用の文字を入れます。
7. 「デプロイ」から「新しいデプロイ」を選びます。
8. 種類は「ウェブアプリ」にします。
9. 実行ユーザーは「自分」、アクセスできるユーザーは「全員」にします。
10. 表示されたWebアプリURLを、透析管理アプリの「Googleスプレッドシート連携」に入力します。

## 食事管理アプリから読む例

飲水量合計を読むURLは次の形です。

```text
GASのWebアプリURL?type=fluidTotal&date=2026-07-14
```

返るデータの例です。

```json
{
  "date": "2026-07-14",
  "totalMl": 650
}
```

`READ_TOKEN` を設定した場合は、URLに `token=読み取りトークン` も付けます。
